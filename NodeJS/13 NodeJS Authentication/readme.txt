Para executar este programa, primeiro você precisa atualizar o backend:

    cd api
    npm install


Depois, precisa instalar o MongoDB Community Edition:

    Baixar de https://www.mongodb.com/try/download/community
    Rodar o Compass (interface gráfica do MongoDB) e criar uma conexão
    Criar o banco de dados "dsw"


Depois, deve executar o nodemon:

    npm install nodemon -g --save
    nodemon


Finalmente, para abrir o front-end, rode os comandos abaixo:

    cd front-end
    npm install
    npm run serve
